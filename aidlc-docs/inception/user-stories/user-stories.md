# ユーザーストーリー

## コンセプト再確認

> **「ぼっちでも、コネなしでも、大学生活を全力で楽しめる。」**

CLaaSのユーザーストーリーは「先輩・友人のコネクションがなくても、AIとプラットフォームが代わりに大学生活を最適化する」という視点で設計されています。麻雀・合コン・ゲームのマッチングから、企業クーポン・就活情報の自動配信まで——ぼっちでも充実した大学生活が送れます。

---

## ペルソナ

### ペルソナA: 田中 蓮（大学2年生）— コネなし・ぼっちの理系学生
- 理工学部・情報工学科
- 先輩のコネクションがなく、楽単情報を得る手段がない
- 空きコマが週に4コマあるが、一緒に過ごす友達がいない
- 教科書代が毎学期2〜3万円かかっていて負担に感じている

### ペルソナB: 佐藤 葵（大学1年生）— 入学したてのぼっち文系学生
- 文学部・英文学科
- 入学したばかりで先輩・友人のコネクションがゼロ
- 「シラバスを読んでも難易度がわからない」「過去問の入手方法がわからない」
- 空きコマで友達を作りたいが、きっかけがない

### ペルソナC: 山田 健太（大学3年生）— 就活と単位の両立に疲弊
- 経済学部
- 就活本番で授業に出る時間が惜しい
- 代返を頼みたいが信頼できる人がいない
- 使い終わった教科書が部屋に山積み

### ペルソナD: 株式会社マイナビ（企業担当者）
- 大学生向けの採用マーケティングを担当
- どの大学のどの学部の学生が暇な時間帯かを知りたい
- 効果的なタイミングで就活情報を届けたい

## ユーザーストーリー一覧

### 😴 最小努力の履修計画ストーリー

---

**US-001: 楽単情報の検索**

```
怠慢を極めたい田中として、
来学期の履修登録前に「情報工学科で最も楽に単位が取れる授業」を検索したい。
そうすることで、最小努力で卒業単位を揃えられる時間割を組める。
```

**受け入れ条件:**
- 大学・学部・授業名で絞り込み検索できる
- 難易度評価・評価方法（出席/レポート/テスト比率）・コメントが表示される
- 「役に立った」数が多い順に並び替えられる

---

**US-002: 過去問の入手**

```
先輩コネゼロの佐藤として、
期末試験前に「英文学概論」の過去問を入手したい。
そうすることで、試験傾向を把握して最小限の勉強で単位を取れる。
```

**受け入れ条件:**
- 授業名・年度・試験種別で検索できる
- PDFのプレビューをダウンロード前に確認できる
- ダウンロードすると投稿者にガクチカクレジットが付与される（共有の動機付け）

---

**US-003: AIによるシラバス自動解析**

```
先輩コネゼロの佐藤として、
シラバスをアップロードするだけで「この授業は出席しなくても単位が取れるか」を知りたい。
そうすることで、試行錯誤なしに最初から楽な授業を選べる。
```

**受け入れ条件:**
- PDFまたはテキストでシラバスをアップロードできる
- 難易度スコア（1-10）と「出席重要度」「レポート負荷」「テスト依存率」が表示される
- 解析結果が楽単情報ページに反映される

---

**US-004: AI自動時間割生成**

```
怠慢を極めたい田中として、
大学・学部・希望単位数を入力するだけで「最も楽な時間割」を提案してほしい。
そうすることで、時間割作りに頭を使わず、最小努力の履修計画が完成する。
```

**受け入れ条件:**
- 最小努力の時間割候補が3パターン以上生成される
- 各候補に総難易度スコア・空きコマ数・根拠が表示される
- 初回（1学期目）は無料で利用できる
- 2学期目以降はガクチカクレジットで利用できる

---

### 💤 空きコマ完全最適化ストーリー

---

**US-005: 空きコマのコンテンツ自動提案**

```
怠慢を極めたい田中として、
水曜3限の空きコマ（90分）に「何をすればいいか」を考えたくない。
そうすることで、考える手間なく最高の暇つぶしができる。
```

**受け入れ条件:**
- 残り時間（90分）に合わせたコンテンツが自動提案される
- 就活モードをオンにすると就活コンテンツが優先される
- 「興味なし」を押すと類似コンテンツの提案が減る

---

**US-006: 近くの施設を探す**

```
怠慢を極めたい田中として、
空きコマに「今すぐ行ける大学近くの映画館」を調べたくない。
そうすることで、検索する手間なく最適な行き先が見つかる。
```

**受け入れ条件:**
- 現在地から近い映画館・美術館・カフェが地図上に自動表示される
- 営業時間・評価・距離が確認できる

---

**US-007: 空きコマで読み終わる漫画・本の提案**

```
怠慢を極めたい田中として、
1時間の空きコマで「読み終わらない本を選んでしまう失敗」をしたくない。
そうすることで、ちょうど読み終わる作品だけが提案される。
```

**受け入れ条件:**
- 空きコマの残り時間以内に読了できる作品だけが提案される
- 平均読了時間が表示される

---

### 🤝 怠慢を支え合うコミュニティストーリー

---

**US-008: 空きコマのマッチング（麻雀・合コン・ゲーム）**

```
ぼっちの佐藤として、
木曜2限の空きコマが被っている人と「麻雀・合コン・スマブラ」をしたい。
そうすることで、自分から声をかける勇気がなくても仲間が見つかる。
```

**受け入れ条件:**
- 同じ大学で空きコマが重複するユーザーが自動表示される
- 麻雀・合コン・ゲーム・スポーツ等のカテゴリでフィルタリングできる
- 相互承認後にチャットが開始される
- コミュニティ（麻雀部・合コン募集等）に参加して仲間を探せる

---

**US-009: 代返の依頼**

```
就活に疲弊した山田として、
就活の面接と授業が被ってしまったので、信頼できる人に代返を頼みたい。
そうすることで、就活を優先しながら出席点を確保できる。
```

**受け入れ条件:**
- 授業名・日時・教室・報酬クレジット数を設定して依頼できる
- Trust_Scoreが高いユーザーが優先的に表示される
- 代返完了後に相互評価でTrust_Scoreが更新される

---

### 🏪 教材循環ストーリー

---

**US-010: 教材の出品**

```
部屋に教科書が山積みの山田として、
使い終わった「マクロ経済学」の教科書を後輩に譲りたい。
そうすることで、捨てる手間なく部屋が片付き、ガクチカクレジットも獲得できる。
```

**受け入れ条件:**
- 教材名・状態・取引形式（無料/クレジット/現金）を設定して出品できる
- 授業と紐付けることで関連する学生に優先表示される
- 無料譲渡の場合は5クレジットが付与される

---

### 💰 ガクチカクレジットストーリー

---

**US-011: クレジットを貯めてさらに楽をする**

```
怠慢を極めたい田中として、
楽単情報や過去問を投稿してガクチカクレジットを貯め、
2学期目以降もAI時間割生成機能を無料で使い続けたい。
そうすることで、課金せずに怠慢ライフを永続できる。
```

**受け入れ条件:**
- 投稿・出席・代返等の行動でクレジットが付与される
- クレジット残高と取引履歴が確認できる
- クレジットを消費してAI時間割生成・プレミアム機能にアクセスできる

---

**US-012: 空きコマ情報を提供してクレジットを得る**

```
怠慢を極めたい田中として、
自分の空きコマ情報を企業に提供するだけでガクチカクレジットを獲得したい。
そうすることで、何もしなくてもクレジットが貯まる。
```

**受け入れ条件:**
- Data_Consentをオンにするだけで10クレジット獲得
- 提供されるのは匿名化・集計データのみ（個人情報は含まない）
- いつでも同意を撤回できる

---

### 🏢 企業向けストーリー

---

**US-013: 企業クーポンが空きコマに自動で届く**

```
ぼっちの田中として、
空きコマに「今すぐ使える映画館のクーポン」や「就活セミナーの案内」が
自分で探さなくても届いてほしい。
そうすることで、企業とのつながりも自動で生まれる。
```

**受け入れ条件:**
- 空きコマの時間帯・興味カテゴリに合わせたクーポンが自動配信される
- クーポン使用でガクチカクレジットが獲得できる
- 映画館・出版社・マイナビ等の企業と連携している

---

**US-014: 空きコマデータの活用（企業側）**

```
マイナビの担当者として、
どの大学のどの学部の学生が何曜日の何限に暇かを知りたい。
そうすることで、学生が暇な時間帯に就活情報を配信して効果を最大化できる。
```

**受け入れ条件:**
- 大学・学部・曜日・時限別の空きコマ人数が確認できる
- データはCSV形式でエクスポートできる
- 個人を特定できる情報は含まれない（5名未満は非表示）

---

## ストーリーマップ

```
ぼっち大学生のCLaaS活用の一日
│
├── 朝: 履修計画（コネなしでも最小努力）
│   ├── US-003: シラバスをAIに解析させる
│   ├── US-004: AI時間割を自動生成させる
│   └── US-001/002: 楽単情報・過去問を検索（先輩不要）
│
├── 昼: 空きコマ（ぼっちでも充実）
│   ├── US-005: コンテンツを自動提案させる
│   ├── US-006: 近くの施設を地図で探す
│   ├── US-007: 読み終わる漫画を提案させる
│   ├── US-008: 麻雀・合コン・ゲームでマッチング
│   └── US-013: 企業クーポンが自動で届く
│
├── 夕: コミュニティ活動
│   └── US-009: 代返を依頼する
│
├── 随時: クレジット活動
│   ├── US-011: 投稿してクレジットを貯める
│   └── US-012: 空きコマ情報を提供してクレジットを得る
│
└── 学期末: 教材循環
    └── US-010: 使い終わった教科書を出品する
```
