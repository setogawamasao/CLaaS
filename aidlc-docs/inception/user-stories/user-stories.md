# ユーザーストーリー

## ペルソナ

### ペルソナA: 田中 蓮（大学2年生）
- 理工学部・情報工学科
- 就活を意識し始めたが、何から始めればいいかわからない
- 空きコマが週に4コマある
- 教科書代が毎学期2〜3万円かかっていて負担に感じている

### ペルソナB: 佐藤 葵（大学1年生）
- 文学部・英文学科
- 入学したばかりで履修選択に不安
- 先輩のコネクションがなく、楽単情報を得る手段がない
- 空きコマで友達を作りたい

### ペルソナC: 山田 健太（大学3年生）
- 経済学部
- 就活本番で時間が足りない
- 代返を頼みたいが信頼できる人がいない

### ペルソナD: 株式会社マイナビ（企業担当者）
- 大学生向けの採用マーケティングを担当
- どの大学のどの学部の学生が暇な時間帯かを知りたい
- 効果的なタイミングで就活情報を届けたい

## ユーザーストーリー一覧

### 🎓 学習支援ストーリー

---

**US-001: 楽単情報の検索**

```
大学2年生の田中として、
来学期の履修登録前に「情報工学科で楽に単位が取れる授業」を検索したい。
そうすることで、GPA維持と就活準備を両立できる時間割を組める。
```

**受け入れ条件:**
- 大学・学部・授業名で絞り込み検索できる
- 難易度評価・評価方法・コメントが表示される
- 「役に立った」数が多い順に並び替えられる

---

**US-002: 過去問の入手**

```
大学1年生の佐藤として、
期末試験前に「英文学概論」の過去問を入手したい。
そうすることで、試験傾向を把握して効率的に勉強できる。
```

**受け入れ条件:**
- 授業名・年度・試験種別で検索できる
- PDFのプレビューをダウンロード前に確認できる
- ダウンロードすると投稿者にガクチカクレジットが付与される

---

**US-003: AIによる履修選択支援**

```
大学1年生の佐藤として、
シラバスをアップロードするだけで授業の難易度を客観的に知りたい。
そうすることで、先輩がいなくても適切な履修選択ができる。
```

**受け入れ条件:**
- PDFまたはテキストでシラバスをアップロードできる
- 難易度スコア（1-10）と根拠（出席要件・試験・レポート等）が表示される
- 解析結果が楽単情報ページに反映される

---

**US-004: AI自動時間割生成**

```
大学1年生の佐藤として、
大学・学部・希望単位数を入力するだけで最適な時間割を提案してほしい。
そうすることで、複雑な履修計画を短時間で完成させられる。
```

**受け入れ条件:**
- 3パターン以上の時間割候補が生成される
- 各候補に難易度スコア・空きコマ数・根拠が表示される
- 生成された時間割を手動で編集できる
- 初回（1学期目）は無料で利用できる

---

### ⏰ 空きコマ活用ストーリー

---

**US-005: 空きコマのコンテンツ提案**

```
大学2年生の田中として、
水曜3限の空きコマ（90分）に何をすればいいか提案してほしい。
そうすることで、空き時間を就活準備や娯楽に有効活用できる。
```

**受け入れ条件:**
- 残り時間（90分）に合わせたコンテンツが提案される
- 就活モードをオンにすると就活コンテンツが優先される
- 「興味なし」を押すと類似コンテンツの提案が減る

---

**US-006: 近くの施設を探す**

```
大学2年生の田中として、
空きコマに大学近くの映画館や美術館を探したい。
そうすることで、キャンパス周辺で充実した時間を過ごせる。
```

**受け入れ条件:**
- 現在地から近い映画館・美術館・カフェが地図上に表示される
- 営業時間・評価・距離が確認できる

---

**US-007: 空きコマで読める漫画・本の提案**

```
大学1年生の佐藤として、
1時間の空きコマで読み終わる漫画を提案してほしい。
そうすることで、中途半端に終わらず満足感を得られる。
```

**受け入れ条件:**
- 空きコマの残り時間以内に読了できる作品が提案される
- 平均読了時間が表示される

---

### 🤝 コミュニティストーリー

---

**US-008: 空きコマのマッチング**

```
大学1年生の佐藤として、
木曜2限の空きコマが被っている人とマッチングしたい。
そうすることで、一緒にランチや勉強ができる友達を作れる。
```

**受け入れ条件:**
- 同じ大学で空きコマが重複するユーザーが表示される
- 共通の興味カテゴリでフィルタリングできる
- 相互承認後にチャットが開始される

---

**US-009: コミュニティへの参加**

```
大学2年生の田中として、
テニスが好きな人のコミュニティに参加したい。
そうすることで、一緒にテニスをする仲間を見つけられる。
```

**受け入れ条件:**
- スポーツ・ゲーム・ボードゲーム等のカテゴリからコミュニティを探せる
- コミュニティ内でメッセージを送受信できる
- 大学限定・公開・招待制の公開設定が選べる

---

**US-010: 代返の依頼**

```
大学3年生の山田として、
就活の面接と授業が被ってしまったので代返を依頼したい。
そうすることで、就活を優先しながら出席点を確保できる。
```

**受け入れ条件:**
- 授業名・日時・教室・報酬クレジット数を設定して依頼できる
- Trust_Scoreが高いユーザーが優先的に表示される
- 代返完了後に相互評価できる

---

### 🏪 マーケットプレイスストーリー

---

**US-011: 教材の出品**

```
大学3年生の山田として、
使い終わった「マクロ経済学」の教科書を後輩に譲りたい。
そうすることで、捨てずに有効活用でき、ガクチカクレジットも獲得できる。
```

**受け入れ条件:**
- 教材名・状態・取引形式（無料/クレジット/現金）を設定して出品できる
- 授業と紐付けることで関連する学生に優先表示される
- 無料譲渡の場合は5クレジットが付与される

---

**US-012: 教材の入手**

```
大学1年生の佐藤として、
「英文学概論」で使う教科書を安く入手したい。
そうすることで、教材費を節約できる。
```

**受け入れ条件:**
- 授業名・状態・取引形式で絞り込み検索できる
- 申し込み後に出品者とチャットで受け渡し方法を調整できる
- 取引完了後に相互評価できる

---

### 🏢 企業向けストーリー

---

**US-013: 空きコマデータの活用**

```
マイナビの担当者として、
どの大学のどの学部の学生が何曜日の何限に暇かを知りたい。
そうすることで、効果的なタイミングで就活情報を配信できる。
```

**受け入れ条件:**
- 大学・学部・曜日・時限別の空きコマ人数が確認できる
- データはCSV形式でエクスポートできる
- 個人を特定できる情報は含まれない（5名未満は非表示）

---

**US-014: クーポンの発行**

```
映画館の担当者として、
大学生向けにクーポンを発行し、来店を促したい。
そうすることで、平日昼間の空席を埋められる。
```

**受け入れ条件:**
- クーポン内容・有効期限・付与クレジット数・発行数上限を設定できる
- ユーザーの興味カテゴリ・空きコマに合わせて配信される
- 使用状況をダッシュボードで確認できる

---

### 💰 ガクチカクレジットストーリー

---

**US-015: クレジットの獲得と活用**

```
大学2年生の田中として、
楽単情報や過去問を投稿してガクチカクレジットを貯め、
2学期目以降もAI時間割生成機能を使い続けたい。
そうすることで、課金せずにアプリの全機能を使い続けられる。
```

**受け入れ条件:**
- 投稿・出席・代返等の行動でクレジットが付与される
- クレジット残高と取引履歴が確認できる
- クレジットを消費してプレミアム機能にアクセスできる
